Whisper STT Stack — Architecture

One fast speech-to-text server on the LAN, serving multiple clients

Clients

  • • voxtype (push-to-talk) on yogaman
  • • Browser demo page (mic needs HTTPS)
  • • Any OpenAI-compatible client
  • • Endpoint: http://slug:8081/v1

Server (slug)

  • • OpenAI proxy :8081 (only public face)
  • • EN on iGPU Vulkan :8083 (~5x faster)
  • • SV on CPU :8082 (kb-whisper-large)
  • • ufw allows 8081; backends loopback-only

Notes

  • • NPU (XDNA2) is Windows-only for whisper
  • • iGPU Vulkan is the Linux accelerator
  • • nginx :8443 optional, for browser mic
  • • systemd units auto-start on boot
